# momentum

🏃 Run Log Tracker
A full-stack Blazor app to track running progress with manual data entry and interactive charts.

✨ Features
Log runs via a user-friendly form.
Data Visualization: Dynamic line charts (MudBlazor) show pace/distance trends.
Dockerized SQL Server: Local database for easy development.
Future Plans: Fitbit API integration + user authentication.

🛠️ Tech Stack
Frontend	Blazor (C#), MudBlazor (UI components)
Backend:	 C#, Entity Framework,	SQL Server (Docker container)
Dev Tools:	Git, VS Code, Docker
